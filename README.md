# Extra Trees Classifier + FastICA - Tarea Investigativa

Implementación de modelos de Machine Learning supervisado y no supervisado con exportación a React.

## Modelos Implementados
- **Supervisado:** Extra Trees Classifier
- **No Supervisado:** FastICA (Análisis de Componentes Independientes)

## Características
- Interfaz interactiva con Streamlit
- Métricas de evaluación completas
- Visualizaciones con Plotly
- Exportación a JSON para React
- Modelos serializados en .pkl

## Instalación
```bash
pip install -r requirements.txt
streamlit run app.py
